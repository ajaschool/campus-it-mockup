# 04. 멘토 · 상품

## 11-2. 멘토 프로필

### 노출 항목
- 학교/단과대/학과
- **입시 연도** (예: "24학번") — 또래 근접성 신뢰 신호. 멘티가 "나와 비슷한 시기에 준비한 선배"임을 즉시 인식하도록 카드·상세 모두 표시
- 입시 경험 (유형: 수시/정시/편입)
- 멘토링 가능 주제
- 대표 키워드
- 자기소개
- 가격
- 상담 시간
- 후기/평점
- 누적 멘토링 수
- 인증 여부
- **상담한 학생 학년 분포 배지** (Week 5+ Low 우선순위 — §W5-badge 참고)

### 중요 포인트
- 프로필 입력은 자유 서술형이 아니라 **구조화된 입력 방식** 지원
- 전환율을 위해 **후기/인기/상담 분야를 상단에 배치**
- 입시 연도는 가입 단계(step1-basic)에서 필수 입력. `mentors` 테이블 `admission_year INT` 컬럼으로 저장. 재심사 필수 필드로 분류.

### §W5-badge — 상담한 학생 학년 분포 배지 (Week 5+ Low 우선순위)

멘토 프로필 카드에 "고1 30% / 고2 50% / 고3 20%" 형태의 텍스트 배지를 노출하는 기능. 학부모·학생이 "내 아이/나와 비슷한 학년을 많이 상담한 멘토"를 빠르게 식별하는 데 도움.

- **데이터**: `bookings` 테이블의 `mentee_grade` 기반 집계. 완료된(`COMPLETED`) 예약 기준
- **집계 주기**: 일 1회 배치 또는 실시간 쿼리(트래픽 적을 때는 쿼리, 부하 시 배치 전환)
- **노출 조건**: 완료 예약 10건 이상인 멘토에게만 표시 (10건 미만은 배지 미노출)
- **구현 포인트**: `mentor_grade_distribution` 뷰 또는 정기 집계 테이블. 프론트는 멘토 상세 API 응답에 `gradeDistribution: { high1: 30, high2: 50, high3: 20 }` 필드 추가
- **우선순위**: Low — 후기/평점 시스템 안정화 후 도입. MVP 4주에는 포함하지 않음

### 프로필 수정 시 재심사 정책

프로필 항목을 **재심사 필수**와 **자유 수정** 두 그룹으로 구분.

| 구분 | 항목 |
|---|---|
| **재심사 필수** (관리자 승인 전까지 기존 값 유지 + 수정 검토중 배지) | 학교 · 단과대 · 학과 · 학년 · 입시 유형 |
| **자유 수정** (즉시 반영) | 닉네임 · 자기소개 · 대표 키워드 · 상담 가능 주제 · 프로필 이미지 · 가격 (관리자 설정 범위 내) |

**운영 규칙**
- 재심사 필수 항목 수정 시 증빙서류 재제출 요구 가능
- 재심사 대기 중에는 신규 예약은 그대로 받음 (기존 값 기준)
- 관리자 반려 시 기존 값 그대로 유지

### 프로필 공개 범위

| 항목 | 기본 공개 | 옵션 |
|---|---|---|
| 이름 | **이니셜 (예: 김OO)** | 멘토 옵트인 시 실명 전체 공개 가능 |
| 학교 | 전체 공개 | — |
| 단과대 · 학과 | 전체 공개 | — |
| 학년 · 입시 유형 | 전체 공개 | — |
| 증빙 서류 사본 | **비공개** (관리자만 열람) | 공개 옵션 없음 |
| 연락처 · 외부 SNS | **노출 금지** (docs/11 §13-5 참조) | — |

## 11-3. 멘토링 상품

멘토는 플랫폼 정의 범주 내에서 상품 등록 가능.

### 필수 항목
- 상품 제목
- 상담 카테고리
- 대상 학년
- 상담 설명
- 진행 시간
- 가격
- 진행 방식
- 상담 전 준비사항
- 상담 후 기대 효과
- **사전 질문 0~10개** (docs/08 §11-8 참고) — 멘티가 결제 시점에 답변하는 멘토 커스텀 질문

### MVP 규칙
- 30분 기본
- 지나치게 자유로운 프로그램 개설은 제한
- **문제 해결형 명칭** 사용 권장 — 등록 화면에서 아래 탬플릿 3종 + 자율 작성 선택 제공

### 상품 제목 탬플릿 (MVP)

멘토 상품 등록 시 제목 입력 UX에서 3가지 탬플릿 중 선택하거나 자율 작성할 수 있도록 제공.
탬플릿 선택 시 `[대상]`·`[분야]` 자리에 입력값을 채워 제목 자동 완성.

| # | 탬플릿 형식 | 예시 |
|---|---|---|
| T1 | `[대상] [분야] 멘토링` | 고1 문과 학생부 종합 전략 멘토링 |
| T2 | `[진로/전공] 진학 Q&A — [입시 유형]` | 의대 진학 Q&A — 수시 학종 집중 |
| T3 | `[고민 키워드] 해결 상담` | 공부 루틴 · 슬럼프 탈출 해결 상담 |
| 자율 | 직접 작성 | (자유 입력, 100자 이내) |

**구현 포인트**
- 프론트 `step3-product.tsx` 탬플릿 3개를 카드 형태로 배치, 선택 시 제목 input에 자동 채움
- "직접 작성" 선택 시 빈 input으로 전환
- 탬플릿을 API·DB에서 관리하지 않고 프론트 상수로 하드코딩 (MVP 단순화)
- 기존 "문제 해결형 명칭" 예시 3개 툴팁은 탬플릿 UI로 대체

### 가격 책정 정책

멘토가 직접 가격을 설정하는 자율 책정 모델. 플랫폼은 관리자가 설정한 최저가~최고가 범위만 강제하고, 그 안에서 멘토가 자유롭게 결정한다.

| 항목 | 값 | 비고 |
|---|---|---|
| 최저가 (강제) | **관리자 설정값** (`platform_settings.price_min`) | 런칭 전 대표 확정. 플랫폼 설정으로 관리, 변경 시 신규 상품부터 적용 |
| 최고가 (강제) | **관리자 설정값** (`platform_settings.price_max`) | 런칭 전 대표 확정. 동일 적용 원칙 |
| 권장가 (UI 표시용) | **29,900원** | 상품 등록 화면에서 가격 input 기본값(placeholder)으로 표시. 강제 아님. "권장 ₩29,900" 레이블로 안내 |
| ~~신규 멘토 floor 락~~ | **폐지** (2026-05-04) | 관리자 설정 최저가로 품질 하한 보호. 별도 락 불필요. docs/11 ADR-003 |
| 가격 변경 주기 | 최대 주 1회 | 덤핑·급등 방지 |
| 결제 시점 가격 | 예약 신청 시점 고정 | 결제 대기 중 멘토가 가격 변경해도 신청된 가격 유지 |
| 가격 노출 UX | 멘토 카드/상세에 멘토 설정가 노출 | 가격 분포 기준선은 실데이터 쌓인 후 W5+ |

**철학**: 더 받고 싶은 멘토는 퀄리티로 증명하고, 시장이 가격과 만족도를 자연스럽게 조정하도록 유도. 플랫폼은 운영 범위(최저~최고가)만 보장하고 세부 책정은 멘토 자율.

#### History
- 2026-04-28 · 권장 상한 ₩99,900은 advisory only(강제 X) 명시 — docs/11 ADR-002
- 2026-05-04 · **가격 모델 전면 전환** — 강제 floor(₩29,900) 폐지. 관리자 설정 최저가·최고가 범위 내 멘토 자율 조정. ₩29,900은 권장가(UI 기본값)로 변경. 신규 멘토 floor 락 폐지. docs/11 ADR-003

### 상품 수정 · 삭제 정책

| 상품 상태 | 가능한 행위 |
|---|---|
| `ACTIVE` | 제목 · 설명 · 가격 · 키워드 등 수시 수정 가능 (관리자 설정 범위 유지). 수정은 **신규 예약부터 반영**, 이미 확정된(`CONFIRMED`) 예약은 신청 시점 가격·조건 그대로 유지 |
| `ACTIVE` → `PAUSED` | 멘토가 일시 중지. 탐색 미노출, 이미 확정된 예약은 그대로 진행 |
| `PAUSED` → `ACTIVE` | 재개 |
| `ACTIVE`·`PAUSED` → `DELETED` (soft delete) | **예정 예약(`REQUESTED`/`CONFIRMED`) 0건일 때만 허용**. 잔여 예약 있으면 먼저 `PAUSED`로만 가능 |

**운영 규칙**
- 삭제는 물리 삭제가 아닌 `status = DELETED`, 탐색·검색에서 제외 (과거 예약 레퍼런스 유지)
- 가격 수정 주기는 §가격 책정 정책의 "최대 주 1회" 규칙 적용
- 수정 이력은 `mentoring_product_revisions` 테이블(Week 5+)에서 관리 권장

### 예시
- 고1 문과 학생 전공 선택 상담
- 내신 공부 루틴 점검 멘토링
- 컴퓨터공학과 진학 Q&A
- 학부모 대상 대학 현실 상담
