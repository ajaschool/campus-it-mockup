# 10. UX·UI · 고객지원

## 12. UX/UI 요구사항

### 12-1. 메인 화면

**핵심 목적**: 전환

**필수 섹션**
- 인기 멘토
- 대학별 탐색
- 단과대/주제별 탐색
- 후기 좋은 멘토
- 지금 많이 찾는 고민 테마

### 12-2. 인기 멘토 영역

회의에서 중요도로 언급된 영역. 전환율에 직접 영향.

**구성 요소**
- 학교/학과
- 대표 상담 주제
- 후기/평점
- 예약 가능 여부
- 가격
- 빠른 신청 CTA

### 12-3. 프로필 입력 UX

멘토가 쉽게 입력하도록 구조화.

- 학교
- 전공
- 전형
- 상담 가능 주제
- 소개 문장 템플릿
- 대표 키워드
- 시간표

### 12-4. 멘토링 플로우 템플릿

30분 기준 표준 구조.

| 구간 | 내용 |
|---|---|
| 0~5분 | 상황 파악 |
| 5~20분 | 핵심 상담 |
| 20~27분 | 질문 응답 |
| 27~30분 | 요약/액션 제안 |

멘토 운영 품질 편차를 줄이기 위한 가이드로 활용.
