# 12. KPI · 로드맵

## 14. KPI

### 초기 운영 KPI
- 검증 완료 멘토 수
- 등록 멘토 수
- 등록 상품 수
- 첫 예약 전환 수
- 예약 확정률
- 알림 확인율

### 품질 KPI
- 평균 평점
- 후기 작성률
- 시간 준수율
- 취소율
- 클레임율

### 수익 KPI
- 총 결제 건수
- 총 거래액
- 멘토 1인당 평균 완료 건수
- 플랫폼 수수료 매출
- 재구매율

### 전략 KPI
- 3개 대학 카테고리 대표성 확보율
- 단과대 커버리지
- 인기 멘토 영역 클릭률
- 탐색→신청 전환율

## 15. 로드맵

### P0 — 핵심 거래 구조
- 멘토 등록/검증
- 프로필 노출
- 상품 등록
- 탐색
- 신청/결제
- 카톡 알림
- 후기/정산
- 관리자 페이지

### P1 — 운영 최적화
- 검색/필터 고도화
- 예약 UX 개선
- 인기 멘토 추천 로직
- 멘토링 카테고리 확장
- 멘토 운영 가이드 강화

### P2 — 고도화
- 자동 추천
- CRM 연동
- 앱 푸시
- 멘토 커뮤니티
- 고급 데이터 리포트
