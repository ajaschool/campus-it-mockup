## 1. 시장 구조와 우리의 포지션

현재 시장은 두 극단으로 양극화되어 있고, 그 사이 공백에 aja-mentoring이 위치한다.

| 경쟁사 | 가격 | 형태 | 구조적 한계 |
|---|---|---|---|
| 메가스터디 명문대 멘토링 | 무료 | 비동기 Q&A 게시판 | 1:1 심층 불가, 외부 사용자 접근 한계 |
| 이투스 E-솔루션 | 무료 | 챗형 Q&A | 단편적 Q&A, 표준화 어려움 |
| 리로스쿨 멘토링 | 무료 (B2B2C) | 포트폴리오 + Q&A | 리로스쿨 미도입 학교 이용 불가, 수익모델 부재 |
| 수시레시피/정시레시피 | 30만원~, 패키지 200만원+ | 전문 컨설턴트 1:1 | 가격 장벽, 맨파워 의존 확장 한계 |
| **aja-mentoring** | **관리자 설정 범위 내 자율 (권장가 29,900원)** | **1:1 화상 (Zoom/Meet)** | **이 자리가 비어 있음** |

**핵심 가치 등식**: "SKY·의대 대학생 1:1 화상 상담을 권장가 29,900원부터"

## 2. 실질 차별점

### 2-1. 또래 근접성 (Peer Proximity)

수시레시피 전문 컨설턴트가 줄 수 없는 가치다.

- "2~3년 전 나와 똑같은 상황이었던 선배"의 공감과 현실적 조언
- 학생이 원하는 것은 전문가의 당위적 조언이 아니라 "선배는 실제로 어떻게 했는지"
- 대형 플랫폼이 가격/규모로 공략해도 복제하기 어려운 속성

**서비스에 반영하는 방법**
- 멘토 카드에 **입시 연도** 표시 ("24학번") → 멘티가 "나와 비슷한 시기 준비" 즉시 인식
- 후기에서 "고2 겨울방학에 비슷한 고민이었는데..." 형태의 경험담 노출 우선

### 2-2. 시장 가격 공백 정확히 점령

학부모 WTP 분석:

- 1회 5~10만원 → "시험 삼아 받아볼 의향" 다수
- 월 20만원 내외 → 심리적 마지노선
- 권장가 29,900원 → 학원 1회 수강료와 비교해 수용 가능 인식 (강제 floor 아니지만 시장 기준선 역할)

무료(메가/이투스)와 고가(수시레시피) 사이 중간 포지션은 신뢰가 뒷받침될 때 폭발적으로 성장할 수 있다.

### 2-3. 학부모-학생 이중 의사결정 구조

경쟁사 대부분은 학생 단일 타겟 또는 학부모 단일 타겟이다.

- 실제 결제 권한자(학부모)가 플랫폼에서 직접 멘토를 탐색·예약
- 학부모 페르소나 전용 신뢰 신호: 인증 배지, 누적 상담 수, 검증된 학교 정보
- 학생 페르소나 전용 신뢰 신호: "나와 비슷한 고민" 후기, 또래 공감형 멘토 소개

구매 결정 마찰을 낮추는 구조적 설계. 경쟁사가 쉽게 복제할 수 없는 이유는 두 페르소나를 위한 UX 톤·신뢰 신호 배치가 완전히 달라야 하기 때문이다.

### 2-4. PRD 정책이 사용자 가치로 전환되는 포인트

| 정책 | 사용자가 느끼는 가치 |
|---|---|
| 결제 시점 가격 고정 | "신청 후 가격 바뀌는 일 없음" → 소비자 보호, 신뢰 |
| 사전 질문지 강제 (0~10개) | 상담 전 준비된 대화 → 품질 표준화. 수시레시피가 전문 컨설턴트 2인조로 구현하는 것을 구조로 달성 |
| 자동 확정 모델 | "신청했는데 묵묵무답" 없음 → 작은 플랫폼에서 이탈률 낮추는 핵심 |
| 취소/환불 3단계 정책 | 멘토·멘티 모두 예측 가능 → 신뢰 기반 거래 |
| 관리자 설정 최저가 | cold-start 품질 덤핑 방지 → 플랫폼 전체 신뢰도 유지 (신규 멘토 floor 락은 ADR-003으로 폐지) |

### 2-5. 아자스쿨 브랜드 신뢰 자산

신규 플랫폼의 가장 큰 장벽은 "이 사이트 믿을 수 있어?"다.

- 기존 아자스쿨 쇼핑몰 회원 기반 → 콜드 스타트 신뢰 공백 일부 해소
- 특히 학부모 고객군에서 브랜드 인지도 전환율에 직접 영향

## 3. MVP에서 경쟁력 강화에 직접 기여하는 기능

### 3-1. 멘토 입시 연도 표시 (Week 2 IN)

- 멘토 가입 필수 입력 (`admission_year`)
- 카드·상세에 "24학번" 형식으로 표시
- 구현 비용: 프론트 select 1개 + 백엔드 컬럼 1개 + 표시 로직
- 상세: `docs/04 §노출 항목`, `plans/week2-mentor.md Day 7`

### 3-2. 상품 제목 탬플릿 (Week 2 IN)

탬플릿 3종 + 자율 작성으로 검색·탐색에 유리한 명칭 자동 유도.

| 탬플릿 | 형식 |
|---|---|
| T1 | `[대상] [분야] 멘토링` |
| T2 | `[진로/전공] 진학 Q&A — [입시 유형]` |
| T3 | `[고민 키워드] 해결 상담` |
| 자율 | 직접 작성 |

- 구현 비용: 프론트 상수 하드코딩, API 연동 불필요
- 상세: `docs/04 §상품 제목 탬플릿`, `plans/week2-mentor.md Day 8`

### 3-3. 상담한 학생 학년 분포 배지 (Week 5+ Low)

멘토 프로필에 "고1 30% / 고2 50% / 고3 20%" 형태 배지 표시.

- 완료 예약 10건 이상 멘토에게만 노출
- `bookings.mentee_grade` 집계 기반
- MVP 미포함. 후기/평점 시스템 안정화 후 도입
- 상세: `docs/04 §W5-badge`

## 4. 리스크 및 주의사항

### 4-1. 50:50 수수료 → 멘토 이탈 리스크

권장가 29,900원 기준 멘토 실수령 14,950원/30분. 시장 평균(크몽 20~40%, 인프런 20~30%)보다 높다. 멘토가 더 높은 가격을 설정하면 수령액이 늘어나는 구조이므로, 자율 가격 범위 내에서 해결 가능.

**대응**: 멘토 가입 시 수익 시뮬레이터 제공 ("₩50,000 상품 주 3회면 월 수령 XX만원") → 숫자 투명화로 부정적 인식 전환. 플랫폼이 제공하는 가치(신규 멘티 발견, 정산 자동화, 예약 관리, 브랜드 신뢰)를 온보딩에서 명시.

### ~~4-2. 신규 멘토 floor 락~~ — ADR-003으로 폐지 (2026-05-04)

관리자 설정 최저가로 품질 하한을 보호하므로 별도 락 불필요. 신규/기존 멘토 동일 가격 범위 적용.

### 4-3. 첫 거래 신뢰 공백

오픈 직후 후기·거래 이력 없는 상태.

**대응**: 오픈 전 시드 리뷰 10~20건 생성 (마케팅팀 베타 테스터 운영). 개발 태스크가 아닌 운영 태스크.

### 4-4. 멘토 시험기간 활동 축소

수요가 몰리는 시즌(5~6월, 11~12월)에 멘토 공급이 줄어드는 역상관 구조. 멘토 풀 지속 확대만이 유일한 대응.

## 5. 핵심 마케팅 메시지 방향

> "SKY·의대 선배와 1:1 화상 상담 — 권장가 29,900원부터"

세 가지 메시지가 모든 화면에서 일관되어야 한다:

1. **또래 신뢰**: "2~3년 전 나와 같은 고민을 했던 선배"
2. **접근성**: "과외비보다 싼 1:1 상담"
3. **검증**: "아자스쿨이 직접 확인한 멘토"

진입 화면 → 탐색 → 멘토 카드 → 상세 → 예약까지 이 메시지가 끊기지 않아야 전환율이 유지된다.
