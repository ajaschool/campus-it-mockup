# 17. 프로젝트 KPI 통합 정리

> 기준일: 2026-04-30  
> 목적: 아자스쿨 온라인 멘토링 MVP의 개발·마케팅·운영 공통 KPI를 한 문서에서 관리한다.  
> 근거: `docs/01-product.md`, `docs/12-kpi-and-roadmap.md`, `plans/4-week-mvp-plan.md`

## 1. KPI 원칙

아자스쿨 멘토링 MVP의 KPI는 단순 방문자 수보다 **검증된 멘토 공급**, **문제 해결형 탐색**, **예약·결제 전환**, **상담 품질**을 우선한다.

초기 홍보는 "SKY 멘토 보유"를 신뢰 메시지로 활용하되, 장기적으로는 **단과대·전공별 대표 멘토풀**을 확보하는 방향으로 확장한다. 따라서 전체 KPI는 다음 5개 축으로 본다.

| 축 | 핵심 질문 |
|---|---|
| 공급 KPI | 상담 가능한 검증 멘토가 충분하고 다양하게 확보됐는가 |
| 유입 KPI | 타겟 학생·학부모가 서비스에 들어오고 있는가 |
| 전환 KPI | 탐색한 사용자가 실제 예약·결제까지 가는가 |
| 품질 KPI | 상담이 안정적으로 진행되고 만족도가 유지되는가 |
| 수익 KPI | 거래액과 플랫폼 수익이 반복 가능한 구조로 만들어지는가 |

## 2. 단계별 목표

| 단계 | 목적 | 기간 | 핵심 목표 |
|---|---|---|---|
| 내부 MVP | E2E 해피 패스 검증 | 4주 개발 기간 | 가입→예약→결제→상담→후기 동작, 멘토 26명 활용 |
| 소프트 런칭 | 실제 수요·전환 검증 | 런칭 후 1~2개월 | 검증 멘토 50명+, 첫 예약/결제 데이터 확보 |
| 초기 성장 | 반복 수요와 카테고리 검증 | 런칭 후 3~6개월 | 멘토 100명, 단과대별 대표성, 재구매 지표 확보 |

## 3. North Star KPI

### 유료 완료 상담 수

가장 중요한 단일 지표는 **유료 완료 상담 수**다.

| 항목 | 정의 |
|---|---|
| 지표명 | 유료 완료 상담 수 |
| 계산 | 결제 완료 후 상담이 `COMPLETED` 상태가 된 건수 |
| 이유 | 단순 가입·방문이 아니라 수요, 공급, 결제, 상담 품질이 모두 연결되어야 증가 |
| 1차 목표 | 소프트 런칭 첫 달 유료 완료 상담 30건 |
| 2차 목표 | 런칭 후 3개월 내 월 유료 완료 상담 100건 |

보조 North Star로는 **상담 완료 후 후기 작성 수**를 함께 본다. 후기 데이터가 쌓여야 검색·노출·신뢰가 개선된다.

## 4. 공급 KPI

### 4-1. 멘토풀 현황

2026-04-28 기준 활동 가능 멘토는 **26명**이다.

| 대학 | 인원 | 의대 | 사범·교육 | 이공계 | 인문·사회 |
|---|---:|---:|---:|---:|---:|
| 서울대 | 8 | 1 | 4 | 1 | 2 |
| 고려대 | 7 | 2 | 0 | 1 | 4 |
| 연세대 | 8 | 3 | 0 | 2 | 3 |
| 부산대 | 1 | 0 | 0 | 1 | 0 |
| 전남대 | 2 | 1 | 0 | 0 | 1 |
| **합계** | **26** | **7** | **4** | **5** | **10** |

현재 강점은 **SKY 23명(88%)**, **의대 7명**, **사범·교육 4명**이다. 부족 영역은 **약대·치대·한의대 0명**, **예체능계 0명**, **자연과학·공학 백업 부족**, **비SKY 주요대학 다양성 부족**이다.

### 4-2. 멘토풀 목표

| KPI | 현재 | 런칭 전 목표 | 6개월 목표 |
|---|---:|---:|---:|
| 검증 완료 멘토 수 | 26명 | 50명+ | 100명 |
| 활동 가능 멘토 수 | 26명 | 45명+ | 80명+ |
| 등록 상품 수 | 미측정 | 멘토당 1개 이상 | 멘토당 평균 2개 |
| 멘토당 주간 슬롯 | 3~4개 가정 | 3개 이상 | 3~5개 |
| 주간 상담 capacity | 78~104건 | 150건+ | 240건+ |

### 4-3. 단과대 커버리지 KPI

정식 런칭 전에는 학교 브랜드보다 **단과대별 선택지**가 중요하다. 마케팅팀 모집 KPI는 아래 기준으로 관리한다.

| 계열 | 런칭 전 최소 | 안정 목표 | 모집 우선도 |
|---|---:|---:|---|
| 의약·보건계 | 8명 | 10명 | 최상 |
| 공과·IT계 | 8명 | 10명 | 상 |
| 자연과학계 | 5명 | 8명 | 상 |
| 인문·사회·상경계 | 8명 | 10명 | 중 |
| 사범·교육계 | 5명 | 8명 | 중 |
| 예체능계 | 5명 | 10명 | 상 |
| 특수·융합·기타 | 3명 | 4명 | 하 |

운영 기준:
- 핵심 6개 계열 중 5개 이상에서 최소 5명 확보
- 단일 계열 비중 25% 초과 시 신규 모집 우선순위 조정
- SKY 비중은 초기 신뢰 확보에 활용하되, 6개월 목표는 60~70% 이하
- 단일 학과 멘토 1명만 있는 카테고리는 백업 멘토 확보 전까지 광고 소재에서 과도하게 강조하지 않음

### 4-4. 1차 추가 모집 목표

현재 26명에서 런칭 전 50명+까지 가려면 최소 24명 추가 모집이 필요하다.

| 우선순위 | 추가 모집 목표 | 이유 |
|---|---:|---|
| 약대·치대·한의대 | 5명 | 의대 다음 수요 대응, 현재 0명 |
| 공과·IT | 5명 | 컴공·AI·전기전자·기계 수요 대응 |
| 예체능 | 5명 | 현재 0명, 디자인·미대·체대 진로 탐색 차별화 |
| 자연과학 | 4명 | 생명·화학·수학·물리 등 이공계 일반 보강 |
| 사범·교육 | 2명 | 공부법·학습 루틴 카테고리 강화 |
| 비SKY 주요대학 | 3명 | 카이스트·포스텍·이화·한양·성균관 등 다양성 확보 |

## 5. 마케팅·유입 KPI

### 5-1. 홍보 목표

초기 마케팅의 목적은 대규모 트래픽보다 **예약 가능성이 높은 학생·학부모 유입**을 만드는 것이다.

| KPI | 정의 | 소프트 런칭 목표 |
|---|---|---:|
| 랜딩 페이지 방문자 수 | 홈 또는 캠페인 페이지 UV | 월 3,000명 |
| 멘토 상세 조회 수 | 멘토 상세 페이지 PV | 월 1,000회 |
| 검색·필터 사용 수 | 검색 페이지 진입 또는 필터 클릭 | 월 500회 |
| 상담 CTA 클릭 수 | 예약/상담 신청 버튼 클릭 | 월 150회 |
| 회원가입 전환 수 | 학생·학부모 가입 완료 | 월 100명 |
| 첫 예약 시도 수 | 예약 플로우 진입 | 월 60건 |

### 5-2. 채널별 KPI

| 채널 | 1차 목적 | 주요 KPI |
|---|---|---|
| 학부모 커뮤니티 | 신뢰 기반 유입 | 게시글 클릭률, 상담 CTA 클릭률, 문의 수 |
| 인스타그램·릴스 | 문제 인식과 브랜드 노출 | 프로필 클릭, 랜딩 유입, 저장·공유 수 |
| 블로그·SEO | 전공·입시 고민 검색 유입 | 검색 유입, 체류 시간, 멘토 상세 조회 |
| 학교·동아리 네트워크 | 멘토 모집 | 지원자 수, 검증 통과율 |
| 기존 아자스쿨 채널 | 초기 신뢰와 전환 | 회원 전환율, 예약 전환율 |

### 5-3. 홍보 메시지별 검증 KPI

마케팅팀은 소재를 "학교 브랜드"와 "문제 해결" 양쪽으로 나눠 검증한다.

| 메시지 유형 | 예시 | 성공 기준 |
|---|---|---|
| 학교 신뢰형 | SKY 선배에게 직접 듣는 입시 경험 | 멘토 상세 조회율 높음 |
| 전공 탐색형 | 의대·공대·예체능 선배에게 전공 현실 듣기 | 검색·필터 사용률 높음 |
| 문제 해결형 | 공부법, 학생부, 면접 고민 30분 상담 | 예약 CTA 클릭률 높음 |
| 학부모 안심형 | 검증된 대학생 멘토와 온라인 상담 | 회원가입 전환율 높음 |

초기에는 "SKY"를 훅으로 쓰되, 실제 랜딩과 탐색 구조에서는 **단과대·전공·고민별 선택지**를 전면에 둔다.

## 6. 전환 KPI

| KPI | 정의 | 소프트 런칭 목표 |
|---|---|---:|
| 방문→상세 전환율 | 방문자 중 멘토 상세 조회 비율 | 25%+ |
| 상세→예약 시도 전환율 | 멘토 상세 조회 중 예약 플로우 진입 비율 | 10%+ |
| 예약 시도→결제 완료율 | 예약 시작 후 결제 완료 비율 | 50%+ |
| 결제 완료→상담 완료율 | 결제 완료 후 상담 완료 비율 | 90%+ |
| 상담 완료→후기 작성률 | 완료 상담 중 후기 작성 비율 | 40%+ |
| 첫 구매 전환 수 | 첫 결제 사용자 수 | 월 30명 |

전환이 낮을 때의 해석:
- 상세 조회는 높은데 예약이 낮으면 멘토 프로필·상품·가격·가능시간 문제
- 예약 진입은 높은데 결제가 낮으면 결제 UX·사전질문 부담·가격 저항 문제
- 결제 후 완료율이 낮으면 알림·일정·노쇼 관리 문제

## 7. 품질 KPI

| KPI | 정의 | 목표 |
|---|---|---:|
| 평균 평점 | 후기 평점 평균 | 4.6점+ |
| 후기 작성률 | 완료 상담 중 후기 작성 비율 | 40%+ |
| 시간 준수율 | 정시 입장·진행 비율 | 95%+ |
| 멘토 취소율 | 멘토 사유 취소 비율 | 3% 이하 |
| 멘토 노쇼율 | 멘토 미입장 노쇼 비율 | 1% 이하 |
| 클레임율 | 완료 상담 대비 클레임 접수 비율 | 3% 이하 |
| 환불율 | 결제 완료 대비 환불 비율 | 5% 이하 |

품질 KPI는 초기에 자동화보다 운영 체크가 중요하다. 관리자 대시보드가 완성되기 전에는 예약·취소·클레임을 운영 시트로 병행 관리한다.

## 8. 수익 KPI

현재 가격 정책은 멘토 자율 책정 + 관리자 설정 최저가~최고가 범위 내이며 (29,900원은 권장가), 플랫폼 수수료는 50% 기준이다.

| KPI | 정의 | 소프트 런칭 목표 | 3개월 목표 |
|---|---|---:|---:|
| 총 결제 건수 | 결제 완료 건수 | 월 40건 | 월 130건 |
| 유료 완료 상담 수 | 완료된 유료 상담 | 월 30건 | 월 100건 |
| 평균 결제 단가 | 결제금액 평균 | 50,000원 | 50,000원+ |
| 월 GMV | 총 거래액 | 150만 원+ | 500만 원+ |
| 플랫폼 수수료 매출 | GMV x 50% | 75만 원+ | 250만 원+ |
| 멘토 1인당 완료 건수 | 완료 상담 / 활동 멘토 | 월 1건+ | 월 2건+ |
| 재구매율 | 2회 이상 결제 사용자 비율 | 측정 시작 | 15%+ |

기존 capacity 가정상 26명 멘토가 주 3~4슬롯을 운영하면 주 78~104건까지 공급 가능하다. 초기에는 공급보다 수요 검증이 병목일 가능성이 높다.

## 9. 운영 대시보드 최소 지표

MVP 관리자 화면 또는 운영 시트에서 최소한 아래 지표는 주간으로 확인한다.

| 구분 | 주간 체크 지표 |
|---|---|
| 공급 | 검증 완료 멘토 수, 활동 가능 멘토 수, 계열별 멘토 수, 등록 상품 수 |
| 유입 | 방문자 수, 멘토 상세 조회 수, 채널별 유입 수 |
| 전환 | 예약 시도 수, 결제 완료 수, 상담 완료 수, 후기 작성 수 |
| 품질 | 취소율, 노쇼율, 클레임율, 평균 평점 |
| 수익 | GMV, 플랫폼 수수료 매출, 평균 결제 단가 |

## 10. 주간 리뷰 규칙

매주 마케팅팀 싱크에서 아래 순서로 본다.

1. 멘토풀 현황: 부족 계열과 신규 모집 우선순위 조정
2. 채널 성과: 유입 수보다 예약 CTA 클릭과 예약 시도 수 중심으로 판단
3. 전환 병목: 상세→예약, 예약→결제, 결제→완료 중 어디서 이탈하는지 확인
4. 품질 이슈: 취소·노쇼·클레임이 특정 멘토나 계열에 몰리는지 확인
5. 다음 주 실험: 홍보 메시지, 모집 타겟, 랜딩 구성 중 1~2개만 변경

## 11. 의사결정 기준

| 상황 | 판단 | 액션 |
|---|---|---|
| 방문은 많은데 예약 시도 부족 | 메시지와 실제 상품 연결 약함 | 랜딩을 고민·전공별로 재구성 |
| 특정 계열 상세 조회가 높음 | 수요 신호 있음 | 해당 계열 멘토 추가 모집 |
| 특정 계열 예약 전환이 낮음 | 공급 품질 또는 가격 문제 | 프로필·상품명·가격·가능시간 점검 |
| 결제 후 취소·노쇼 발생 | 운영 품질 문제 | 알림, 리마인드, 멘토 패널티 적용 점검 |
| 후기 작성률 낮음 | 상담 후 루프 약함 | 후기 작성 알림과 작성 UX 개선 |
| 재구매 없음 | 단발 상담에 그침 | 후속 상담 상품, 학년별 로드맵 상품 검토 |

## 12. 현재 결론

현재 26명 멘토풀은 내부 MVP와 소프트 런칭 초반 검증에는 충분하다. 다만 홍보 관점에서는 "SKY 위주"만 강조하면 선택지가 좁아 보일 수 있으므로, 정식 런칭 전 KPI는 **검증 멘토 50명+**, **핵심 계열별 최소 5명**, **예체능·의약·공과·자연과학 보강**으로 잡는다.

마케팅팀의 1차 목표는 많은 트래픽을 만드는 것이 아니라, **멘토 상세 조회 → 예약 시도 → 결제 완료**까지 이어지는 고의도 유입을 찾는 것이다.
