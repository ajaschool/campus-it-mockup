# 08. 품질 · 탐색 보조

## 11-8. 사전 질문지

상담 품질 표준화를 위해 **필수**. **멘토가 미리 질문 작성 → 멘티가 결제 시점 답변** 모델.

### 운영 방식

1. **멘토**: 상품 등록 시 사전 질문 N개 직접 작성 (개수 제한·예시 가이드 제공)
2. **멘티**: 결제 페이지에서 사전 질문 답변 폼 + 결제 폼 함께 노출 (한 페이지 스크롤)
3. **검증**: 모든 필수 질문 답변 미작성 시 결제 진행 불가 (서버 검증)
4. **활용**: 멘토는 확정 즉시 답변 열람 가능 → 상담 준비

### 공통 기본 항목 (플랫폼 제공, 모든 상담 공통)

- 학년 (드롭다운, docs/14 `grade_code`)
- 관심 대학/전공 (자유 입력)
- 현재 성적대 (드롭다운, docs/14 `score_range`)
- 학부모 동석 여부 (체크박스)

### 멘토 커스텀 질문 (상품별)

- 멘토가 상품 등록 시 추가 질문 0~10개 작성 가능 (텍스트 답변형)
- 예: "공부법 멘토링 - 현재 사용 중인 인강·교재가 있다면 알려주세요"
- 각 질문은 필수/선택 플래그 설정 가능

### 알림 정책

- **사전질문 미작성 리마인드 알림 없음** (결제 시점 답변 강제로 미작성 케이스 자체가 없음)
- 답변 누락 시 결제 버튼 비활성

## 11-9. 후기/평점

### 후기 작성 대상
- 상담 완료자(`booking.status = COMPLETED`)만 가능
- 작성 기한: **상담 종료 후 14일**. 이후 자동 마감 (시스템이 작성 불가 처리)

### 항목
- 전반 만족도
- 설명 이해도
- 문제 해결 도움 정도
- 시간 준수
- 재신청 의향
- 텍스트 후기

### 활용
- 인기 멘토 영역
- 검색 정렬
- 추후 멘토 선별/노출 기준

### 수정 / 삭제 / 답글 / 신고

| 행위 | 주체 | 조건 |
|---|---|---|
| **수정** | 작성자(멘티) | 작성 후 **7일 이내**, **최대 1회** 수정 가능. 수정 이력은 DB 보존 |
| **삭제** | 작성자(멘티) | 작성 후 **30일 이내** 본인 삭제 가능. 이후는 관리자 경유 (§13-3) |
| **답글** | 해당 멘토 | 후기당 **1회** 답글 가능. 답글도 동일하게 7일 내 수정 · 30일 내 본인 삭제 |
| **신고** | 양측(멘티·멘토) | 부적절·악성·허위·모욕 후기 신고 가능. 관리자 검토 후 `is_hidden = true` + `hidden_reason` 코드 부여 (docs/14 §8-1) |

#### 운영 원칙
- 수정 가능 기간이 지난 후기는 "수정 기한 만료" 안내 노출
- 멘토는 자기 받은 후기에만 답글 가능. 다른 멘토 후기 접근 불가
- 반복 신고 남발(30일 내 5회 이상 기각) 시 계정 경고
- 후기 삭제 시에도 **집계(`avg_rating`, `review_count`)는 자동 재계산**
- 멘토 답글은 후기와 동일 카드에 인용 형식으로 노출

## 22. 탐색 보조 기능 (v1.1)

### 22-1. 찜(북마크)
- 멘토 상세/카드 UI에서 하트 아이콘
- 마이페이지에서 찜 목록 관리

### 22-2. 최근 본 멘토
- 최근 10명까지 자동 기록
- 비회원도 쿠키 기반 임시 기록 (로그인 후 동기화)

### 22-3. 검색 정렬/필터

**정렬 옵션**
- 인기순 (기본)
- 후기 많은 순
- 평점 높은 순
- 신규 등록 순
- 가격 낮은 순

**필터**
- 예약 가능 시간대 필터 (요일·시간대)
- 가격대 필터
- 단과대/카테고리 필터
